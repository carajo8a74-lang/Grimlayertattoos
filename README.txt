ADMIN FIX (Grimlayer)

Upload these files to /admin/ in your Netlify deploy (replace existing):
- admin/index.html
- admin/config.yml

Requirements:
1) Site linked to a Git repository (Build & deploy → Link to a repository).
2) Identity: Enabled
3) Git Gateway: Enabled (Identity → Settings & usage → Services)
4) Invite your email in Identity → Users and confirm the invite.

Then open: https://grimlayertattoo.com/admin/?nocache=1
