GRIMLAYER — Pasos rápidos (móvil)

1) Sube estas carpetas/archivos a un repo en GitHub.
2) En Netlify: Add new site → Import from Git → elige el repo.
   - Build command: (vacío)
   - Publish directory: .
3) Site settings → Identity → Enable.
4) Identity → Services → Enable Git Gateway.
5) Abre /admin/ e inicia sesión (o invítate).
6) /admin/health.html debe mostrar 200 y 200/204.
